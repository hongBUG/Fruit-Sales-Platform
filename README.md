## 1. 完成了项目初期的环境配置
> 环境构建过程中出现了很多的问题，总结下来有以下几类：
> 1). 自己粗心，导致一些变量名写错，项目初始化的时候得不到正确的文件或者变量；
> 2). 配置的maven包没有导入, 这个没啥可说的，自己根据cause by 的信息，查看自己缺了哪些包；
> 3). 依赖的包冲突  比如mybatis和ibatis包冲突, 这属于历史遗留问题  把ibatids包删掉就好
> 4).  版本冲突，比如Spring和Mybatis的不同版本之间会有冲突
